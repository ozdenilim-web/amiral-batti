# Amiral Battı — Online Multiplayer

11x11 grid, T-şekilli Amiral, 3'er atış, köşe dahil değme yasağı.

## Kurulum

```bash
npm install
npm run dev
```

## Deploy (Vercel)

1. GitHub'a push et
2. Vercel'de "Import Project" → repo seç
3. Deploy

## Nasıl Oynanır

1. Bir oyuncu "Yeni Oda Oluştur" → oda kodu alır
2. Kodu rakibine gönderir
3. İkisi de gemileri yerleştirir
4. Sırayla 3'er atış — 20 hücre vurulunca oyun biter
